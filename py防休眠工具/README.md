# py防休眠工具

## 简介

屏幕活动工具是一款简单易用的Python应用程序，旨在防止电脑因长时间无操作而自动息屏或退出某些程序。通过模拟鼠标移动，该工具可以帮助保持屏幕活跃状态，适用于需要长时间保持屏幕活动的各种场景。

## 功能特点

- **位置选择**：用户可以选择1到5个屏幕上的位置，程序会按顺序移动鼠标到这些位置。
- **间隔时间设置**：用户可以设置每次移动鼠标之间的间隔时间，默认为2秒。
- **重复次数设置**：用户可以设置鼠标移动的总次数，默认为100次。
- **停止功能**：提供一个停止按钮，用户可以随时终止程序的运行。

## 安装

1. 确保已安装Python 3。
2. 安装所需的库：
   ```bash
   pip install pyautogui pynput
   ```

## 使用方法

1. 运行程序：
   ```bash
   python app.py
   ```

2. **添加位置**：
   - 点击“添加位置”按钮。
   - 在屏幕上点击以选择位置，最多可以添加5个位置。
   - 位置选择完成后，程序会自动记录位置坐标。

3. **设置间隔时间和重复次数**：
   - 在“间隔时间(秒)”输入框中输入每次移动鼠标之间的间隔时间。
   - 在“重复次数”输入框中输入鼠标移动的总次数。

4. **开始活动**：
   - 点击“开始”按钮，程序将按照设定的间隔时间和重复次数移动鼠标到指定的位置。

5. **停止活动**：
   - 点击“停止”按钮，程序将停止鼠标移动。

## 示例

假设你需要保持屏幕活跃状态1小时，可以按以下步骤操作：

1. 添加位置：
   - 点击“添加位置”按钮，在屏幕上选择1到5个位置。
   
2. 设置参数：
   - 设置间隔时间为2秒。
   - 设置重复次数为1800次（1小时 = 3600秒 / 2秒 = 1800次）。

3. 开始活动：
   - 点击“开始”按钮，程序将每隔2秒移动鼠标到一个位置，总共移动1800次。
